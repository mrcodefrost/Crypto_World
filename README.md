
# Crypto World

Crypto World is a Flutter web application designed to track various cryptocurrencies, NFTs, and exchanges. Built using the Flutter web framework with Dart language, Crypto World offers a convenient and user-friendly interface for users to stay updated on the latest information in the world of cryptocurrencies.
## Features

- **Cryptocurrency Tracking**: Stay informed about the latest prices, market caps, and trends for a wide range of cryptocurrencies.

- **NFT Monitoring**: Explore the world of non-fungible tokens (NFTs) and keep track of their prices and availability.

- **Exchange Information**: Access real-time data from different cryptocurrency exchanges, including trading volumes and liquidity.

- **Responsive Design**: Enjoy a seamless experience across various devices and screen sizes, thanks to Crypto World's responsive and adaptive design.

## Technologies Used

- **Flutter Web**: The app is built using the Flutter web framework, allowing for cross-platform compatibility and fast development.

- **Dart Language**: Dart is used as the primary programming language for building the application logic and user interface.

- **GetX**: GetX is utilized for state management, providing a simple and efficient way to manage the app's state and data flow.

- **CoinGecko API**: Data is fetched from the CoinGecko API, a comprehensive cryptocurrency data platform, to provide accurate and up-to-date information to users.

## Deployment

Crypto World is deployed on Firebase Hosting, making it accessible to users worldwide through any modern web browser. There's no need for installation, as the app can be accessed directly via its web URL.
